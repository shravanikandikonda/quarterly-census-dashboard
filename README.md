# Quarterly Census Dashboard

This project is an interactive dashboard built using Streamlit and Pandas to analyze Quarterly Census Employment and Wage data.

The dashboard allows users to upload a CSV file and explore employment, wage, industry, and regional trends through interactive charts and filters.

## Features

- Employment trend analysis
- Year-based filtering
- Industry analysis
- Wage analysis
- Interactive data explorer
- Statistical summaries

## Technologies Used

- Python
- Pandas
- Streamlit

## Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
